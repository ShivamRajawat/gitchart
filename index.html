Git 2.x All Platforms Quick Reference
⚙
Setup & Initialization
config · init · clone
Identity
git config --global  user.name "Name"
Set global username
git config --global  user.email "email"
Set global email
git config --list
View all config values
Start a Repo
git init
Init repo in current dir
git init <folder>
Init repo in new folder
git clone <url>
Clone remote repo locally
git clone <url> <dir>
Clone into named folder
Snapshot
git status
Show working tree status
git add <file>
Stage specific file
git add .
Stage all changes
git commit -m "msg"
Commit with message
git commit --amend
Modify last commit
git diff
Unstaged changes diff
git diff --staged
Staged vs last commit
git log --oneline
Compact commit history
git log --graph
Visual branch history
⎇
Branching & Merging
branch · merge · rebase
Branch Management
git branch
List local branches
git branch -a
List all branches (incl. remote)
git branch <name>
Create new branch
git branch -d <name>
Delete merged branch
git branch -D <name>
Force delete branch
git branch -m <old> <new>
Rename branch
Switching
git switch <branch>
Switch to branch
git switch -c <branch>
Create & switch branch
git checkout -- <file>
Discard file changes
Merge & Rebase
git merge <branch>
Merge branch into current
git merge --no-ff <branch>
Merge with merge commit
git merge --abort
Abort conflicted merge
git rebase <branch>
Rebase onto branch
git rebase -i HEAD~N
Interactive rebase last N
git cherry-pick <sha>
Apply specific commit
☁
Remote & Advanced
remote · stash · reset · tags
Remotes
git remote -v
List remotes with URLs
git remote add <name> <url>
Add a new remote
git fetch <remote>
Fetch without merging
git pull
Fetch & merge (fast-forward)
git pull --rebase
Fetch & rebase local commits
git push <remote> <branch>
Push branch to remote
git push -u origin <branch>
Push & set upstream
git push --force-with-lease
Safe force push
Stash & Clean
git stash
Stash working changes
git stash pop
Apply & remove top stash
git stash list
Show all stash entries
Undo & Reset
git revert <sha>
New commit undoing SHA
git reset --soft HEAD~1
Undo commit, keep staged
git reset --hard HEAD~1
Undo commit & discard changes
✦
Power Commands & Workflows
aliases · search · tags · reflog
Search & Inspect
git log --author="name"
Filter log by author
git log --since="1 week"
Commits in time range
git log -S "string"
Commits adding/removing string
git grep "pattern"
Search working tree content
git blame <file>
Show who changed each line
git show <sha>
Inspect a specific commit
Tags & Reflog
git tag
List all tags
git tag <name>
Create lightweight tag
git tag -a <name> -m "msg"
Create annotated tag
git push origin --tags
Push all tags to remote
git reflog
Full history incl. resets
git bisect start
Binary search for bug commit
Useful Aliases
git config --global alias.co checkout
co = checkout
git config --global alias.br branch
br = branch
git config --global alias.st status
st = status
git config --global alias.lg "log --oneline --graph"
lg = visual log
git config --global alias.undo "reset --soft HEAD~1"
undo = soft reset
git config --global alias.wip "!git add -A && git commit -m WIP"
wip = quick save commit
cyan = arg / value
lime = branch / file
amber = remote / ref
red = destructive
purple = SHA / commit
git version 2.x · <angle brackets> = replace with your value
